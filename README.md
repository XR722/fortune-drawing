# 福气抽签 · 今日运势测评

移动端运势测评 H5 页面。用户回答 5 道随机题目后，获得当日福签，展示运势等级、诗词、五维评分及宜忌建议，支持截图保存分享。

## 快速开始

```bash
# 任意静态服务即可运行
python -m http.server 8080
# 或
npx serve .
```

打开 `http://localhost:8080` 即可体验。

## 页面结构

| 页面 | 功能 |
|------|------|
| 首页 | "福"字印章动画、开始抽签按钮 |
| 答题页 | 5 道随机题，选中后滑出→滑入切换动画 |
| 算福动画 | 答完后 3 秒算福过渡：旋转光环 + 脉冲福字 + 粒子 |
| 福签页 | 运势等级、四句诗词、五维评分、宜忌建议 |
| 保存分享 | html2canvas 截图生成图片，长按保存 |

## 题库

- 数量：1000 题（50 个模板 × 5 维度 × 20 轮选项轮转）
- 维度：缘分、事业、财运、健康、运势，每类约 200 题
- 随机：使用 `Math.random()`，每人每次抽 5 题各不相同

题库位置：[`templates` 对象](index.html#L835)，增删改模板即可。

## 福签等级

| 等级 | 触发条件（得分率） |
|------|--------------------|
| 大吉 | ≥ 80% |
| 中吉 | ≥ 60% |
| 小吉 | ≥ 45% |
| 吉 | ≥ 30% |
| 末吉 | < 30% |

每级含 10 条诗词和 10 条宜忌建议，随机展示。

福签数据位置：[`fortuneData` 对象](index.html#L924)。

## 技术栈

纯静态单文件：HTML + CSS + JS，零依赖框架，通过 CDN 引入 `html2canvas` 做截图。

### 设计风格

- 深色玄红底色 + 金色点缀
- Noto Serif SC 衬线字体
- CSS 动画 + 浮动粒子背景
- `prefers-reduced-motion` 降级支持
